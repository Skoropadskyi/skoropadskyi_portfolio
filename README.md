# Skoropadskyi Portfolio

A simple portfolio starter theme built with Vue.js v3 and Tailwind CSS v3.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
